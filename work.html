Bee Cycles

Bee Cycles is a local bike repair shop in which I was a customer few times and would highly recommend due to staffs professionalism. Because of positive experience I've had while being Bee Cycle's customer, this website was built based on imaginary products as at the moment Bee Cycles does not possess big stock of product. This specific website/app was built as a final project while studying as a Full Stack Software Developer at Code Institute. Goal was to build an fully functional e-commerce website. Visitors can create accounts, existing users can log in and view their profile, where their purchases are shown and personal details like delivery address. In case admin permissions are given to users, they would be able to add, edit and delete products. Stripe is implemented as payment system and static files like images and css files are hosted on AWS.

Technologies and Languages used:
  • Python
  • Django
  • Jinja2 - python templating language
  • SQLlite as development database
  • Heroku postgres as deployed database
  • Heroku to host website
  • AWS to host static files
  • Stripe as payment system
  • JavaScript/JQuery
  • HTML/CSS
  • Bootstrap
  • Git/Github to store code
